Descrição do Curso

Este curso apresentará conceitos de web e mostra as partes e funções das tags em um documento HTML. Ensina, também, a formatar uma página, inserir imagens e muito mais! Tudo isso, com explicações simples e de fácil compreensão.

Estrutura do Curso

  1. Módulo 1 - Introdução
  2. Módulo 2 - HTML e a web
  3. Módulo 3 - Criando um documento HTML
  4. Módulo 4 - Formatando um documento em HTML
  5. Módulo 5 - Criando Listas
  6. Módulo 6 - Inserindo Imagens
  7. Módulo 7 - Crinado Links
  8. Módulo 8 - Combinando Tags
  9. Módulo 9 - Usando Imagens Mapeadas
